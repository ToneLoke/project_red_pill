### DEVELOPMENT ENVIRONMENT SETUP

> *Ensure mongodb 3.4 or higher is installed on machine*
1. Run db server locally in terminal

```
mongod
```
2. Install depencies

```
yarn install
```

3. Start development server

```
yarn start
```
